Welkom op onze vernieuwde site!

Ontdek de nieuwste functies en leer hoe je Arya optimaal kunt benutten.


- AryaLM3.1, ons nieuwste taalmodel exclusief voor ingelogde gebruikers. Nog niet ingelogd? Ga dan naar de accountpagina om van dit nieuwe taalmodel gebruik te kunnen maken! Als je ingelogd, heb je automatisch direct toegang tot dit nieuwe taalmodel!

- Nieuwe UI, die een aantal bugs heeft opgelost en een nieuwer en moderner design

- Arya Trainer, om zelf code te schrijven voor Arya. Klik hier voor de Arya Trainer en stuur ons via de knop linksboven van die pagina gelijk jouw code!

Kijk voor alle nieuwe functies op de nieuwspagina. Veel plezier met deze update!

Jouw invoer wordt verzameld voor verbeteringen aan onze taalmodellen. Meer informatie

? Elke keer als de data van Arya wordt uitgebreid, komt er een nieuw taalmodel (LM, Language Model). Je kan tussen taalmodellen kiezen in het oranje selectie-gebied naast het typevak!

Arya heeft ook verschillende functies, zoals normaal chatten en bestanden lezen. Deze functies vindt je rondom het typevak.

Zelf kan je ook meehelpen aan Arya door zelf code te schrijven. Wij hebben hier een speciale tool voor gemaakt, namelijk de Arya Trainer. Hier kan je de invoer invullen (wat de gebruiker moet zeggen) en de uitvoer (antwoord van Arya). Hiervoor hoef je zelf geen code te schrijven. Je krijgt na afloop regels code, deze kan je vervolgens via de contactpagina naar ons sturen.

Arya is mede mogelijk gemaakt door Ragheb, Ibrahim, Harry en Olivier. Kijk op de Creditspagina voor meer informatie.
