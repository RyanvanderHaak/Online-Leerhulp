Welkom bij de verniewde Arya!

Ontdek de nieuwste functies en leer hoe je Arya optimaal kunt benutten.


- AryaLM3.2, ons nieuwste taalmodel! Dit model biedt de beste herkenning ooit en introduceert een nieuwe functie: Actions! Actions zijn acties waarmee je iets kan voltooien. Denk aan "Breng me naar YouTube" en andere linkjes en acties. Deze acties zullen worden uitgebreid in elk nieuw taalmodel van Arya.

Sommige elementen zijn verbeterd met een betere UI voor de beste chat-ervaring!

- Arya Trainer is geüpdatet met nieuwe sneltoetsen om zelf code te schrijven voor Arya. Klik hier voor de Arya Trainer. Dit zijn alle sneltoetsen:

- Ctrl + B, om de vorige regel te kopiëren en dus een extra leesteken of andere verwoording toe te voegen voor betere herkenning.

- Ctrl + Enter, om gelijk de data te trainen.

- Ook worden al jouw regels met zinnen en data nu automatisch opgeslagen in je browser zodat je direct verder kan werken!

Kijk voor alle nieuwe functies op de nieuwspagina of op het YouTube-kanaal. Veel plezier met AryaLM3.2 en alle extra nieuwe functies!

Jouw invoer wordt verzameld voor verbeteringen aan onze taalmodellen. Meer informatie

? Om te zorgen dat Arya steeds slimmer wordt en tekst beter kan begrijpen, brengen wij om de paar maanden updates uit. Elke keer als de data van Arya wordt uitgebreid, komt er een nieuw taalmodel (LM, Language Model). Je kan tussen deze taalmodellen kiezen in het selectie-gebied naast het typevak!

Arya heeft ook verschillende andere functies, zoals Actions om naar een website te gaan en bestanden lezen. Deze functies vindt je rondom het typevak en door het te vragen.

Zelf kan je ook meehelpen aan Arya door zelf code te schrijven. Wij hebben hier een speciale tool voor gemaakt, namelijk de Arya Trainer. Hier kan je de invoer invullen (wat de gebruiker moet zeggen) en de uitvoer (antwoord van Arya). Hiervoor hoef je zelf geen code te schrijven. Je krijgt na afloop regels code, deze kan je vervolgens via de contactpagina naar ons sturen.
